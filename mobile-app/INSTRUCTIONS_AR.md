# كيفية رفع تطبيقك على Vercel

تهانينا على إنشاء تطبيقك باستخدام Next.js! الآن، اتبع هذه الخطوات البسيطة لرفعه على الإنترنت مجانًا باستخدام Vercel.

## المتطلبات
- حساب على [GitHub](https://github.com).
- مشروع Next.js الذي قمنا بإنشائه (`mobile-app`).

---

### الخطوة 1: رفع الكود إلى مستودع GitHub

قبل أن تتمكن من نشره على Vercel، يجب أن يكون الكود الخاص بك موجودًا على GitHub.

1.  **أنشئ مستودعًا جديدًا (New Repository) على GitHub:**
    - اذهب إلى حسابك في GitHub واضغط على زر "New".
    - أعطِ المستودع اسمًا (مثلاً: `my-mobile-app`) واتركه عامًا (Public).
    - لا تقم بتحديد أي خيارات مثل `README` أو `.gitignore` لأن المشروع يحتوي عليها بالفعل.
    - اضغط على "Create repository".

2.  **اربط مشروعك المحلي بالمستودع على GitHub:**
    - افتح الطرفية (Terminal) في مجلد مشروعك (`mobile-app`).
    - قم بتنفيذ الأوامر التالية بالترتيب. **تأكد من استبدال `<رابط المستودع الخاص بك>` بالرابط الذي حصلت عليه من GitHub في الخطوة السابقة.**

    ```bash
    # تهيئة مستودع Git محلي
    git init -b main

    # إضافة جميع الملفات إلى المرحلة
    git add .

    # إنشاء أول commit
    git commit -m "Initial commit: My responsive Next.js app"

    # ربط المستودع المحلي بالمستودع على GitHub
    git remote add origin <رابط المستودع الخاص بك>

    # رفع الكود إلى GitHub
    git push -u origin main
    ```

---

### الخطوة 2: نشر المشروع على Vercel

الآن بعد أن أصبح الكود على GitHub، عملية النشر سهلة جدًا.

1.  **سجل في Vercel:**
    - اذهب إلى [vercel.com](https://vercel.com).
    - اضغط على "Sign Up" واختر "Continue with GitHub". هذا هو الخيار الأسهل لأنه يربط حساباتك تلقائيًا.

2.  **استيراد المشروع من GitHub:**
    - بعد تسجيل الدخول، ستكون في لوحة التحكم (Dashboard).
    - اضغط على زر "Add New..." واختر "Project".
    - ستقوم Vercel بالبحث في مستودعات GitHub الخاصة بك. اختر المستودع الذي أنشأته للتو (`my-mobile-app`).
    - اضغط على "Import".

3.  **ضبط المشروع والنشر:**
    - سيتعرف Vercel تلقائيًا على أن هذا مشروع Next.js وسيملأ جميع إعدادات البناء (Build Settings) بشكل صحيح.
    - **لست بحاجة إلى تغيير أي شيء!** الإعدادات الافتراضية مثالية.
    - اضغط على زر "Deploy".

4.  **انتظر واستمتع!**
    - سيبدأ Vercel في بناء ونشر تطبيقك. تستغرق العملية عادةً دقيقة أو دقيقتين.
    - بمجرد الانتهاء، ستحصل على رابط لموقعك المباشر على الإنترنت (مثل `my-mobile-app.vercel.app`).

---

### الميزة الإضافية: النشر التلقائي (Automatic Deployments)

أفضل جزء في استخدام Vercel مع GitHub هو أنه في كل مرة تقوم فيها بعمل `git push` لفرع `main` الخاص بك، سيقوم Vercel تلقائيًا بإعادة بناء ونشر التغييرات الجديدة. هذا يجعل عملية التحديث سهلة للغاية!
